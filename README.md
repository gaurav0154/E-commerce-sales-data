# E-commerce-sales-data
📊 E‑Commerce Sales Data Analysis Project  

This project focuses on analyzing large‑scale e‑commerce sales data to uncover business insights.  
It includes cleaning raw transactional data, building relationships between orders, customers, and products, and applying measures like revenue, profit, and order count.  
Dashboards highlight top‑selling categories, regional performance, customer segments, and seasonal trends.  
The objective is to help businesses optimize marketing, inventory, and customer experience through data‑driven decisions.  

👉 GitHub Description:  
"E‑Commerce Sales Analysis using Power BI & Excel – A complete end‑to‑end project covering data cleaning, modeling, DAX calculations, and interactive dashboards. Provides insights into revenue trends, customer behavior, product performance, and regional sales to support smarter business strategies."
